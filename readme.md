\## 📘 책 소개



\*\*《데이터로 시작하는 주식투자》\*\*는 파이썬을 활용하여 주식 투자의 정석인 \*\*멀티팩터(Multi-Factor) 전략\*\*을 구현하고, 더 나아가 최신 인공지능 기술인 \*\*LLM(거대 언어 모델)과 MCP\*\*를 투자 프로세스에 접목하는 방법을 다루는 실전 가이드북입니다.



데이터 수집부터 팩터 분석, 종합 지표 생성, 그리고 AI를 활용한 시장 분석까지, 체계적이고 과학적인 투자 방법을 제시합니다.



\### 💡 이 책의 핵심 포인트

1\.  \*\*멀티팩터 전략의 실체화:\*\* 모멘텀, 밸류, 퀄리티 등 핵심 팩터 데이터를 직접 수집하고 분석하여 나만의 강력한 투자 지표를 만듭니다.

2\.  \*\*AI 기술의 투자 접목:\*\* 챗GPT와 같은 LLM과 MCP 기술을 활용하여 시장 분석, 종목 리포트 분석 등 투자 효율을 극대화하는 방법을 배웁니다.

3\.  \*\*파이썬 실전 활용:\*\* 금융 데이터 분석에 꼭 필요한 파이썬(Pandas) 핵심 문법을 익히고, 이를 실전 코스피/코스닥 데이터에 적용합니다.

\## 🛠️ 환경 설정 (Setup)



원활한 실습을 위해 Python 3.8 이상 환경에서 아래 명령어를 통해 필요한 라이브러리들을 설치해주세요.



``` 

\# 저장소 클론

git clone https://github.com/DrFirstLee/python\_investment.git



\# 디렉토리 이동

cd python\_investment

``` 



\*\*주요 사용 라이브러리:\*\* `pandas`, `numpy`, `matplotlib`, `seaborn`, `finance-datareader`, 그리고 LLM 활용을 위한 관련 라이브러리 등.
